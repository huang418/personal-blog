抖音小游戏（ByteDance）构建要点
1. 在 Cocos Creator 构建面板选择平台：ByteDance / 抖音小游戏。
2. 体积限制：
   - 使用分包（subpackages）拆大资源（音频/大图）。
   - 精简首包资源（主场景+核心逻辑），其余资源通过远端下载或分包。
3. 调试与真机测试：
   - 抖音平台有域名与 socket 白名单，线上多人需申请并配置域名。
   - 测试真机时注意帧率、drawcall、纹理压缩（选择合适的格式）。
4. 发布：
   - 构建后使用抖音小游戏管理后台上传（或使用 CLI 自动上传）。
   - 若需要录屏/分享功能，请接入抖音开放能力（SDK）。
